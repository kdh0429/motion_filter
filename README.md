# Dependency
* manif
```bash
git clone https://github.com/artivis/manif.git
cd manif && mkdir build && cd build
cmake ..
make install
```
* toml11
```bash
git clone https://github.com/ToruNiina/toml11.git
cd toml11 && mkdir build && cd build
cmake ..
make install
```
